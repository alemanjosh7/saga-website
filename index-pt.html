Primeiro uso de SAGA

As etapas para usar o SAGA são muito simples, as quais são detalhadas a seguir:

1- Abra o programa.

2- Ser-lhe-á pedido que inicie a primeira utilização, pelo que deverá seguir as instruções que lhe serão fornecidas.

3- Cadastre sua empresa, você pode começar a registrar os containers que você possui.

4- Feito! Agora você pode usar SAGA

Usando SAGA

Passos para registrar

1- Escolha a opção em que deseja se cadastrar, para isso deverá clicar no botão da SideBar que contém a sua escolha.

2- Aqui, procure o botão de registro na parte inferior.

3- Em seguida, preencha as informações solicitadas.

4- Quando terminar, clique em salvar e pronto

Passos para atualizar um dado

1- Escolha a opção que deseja se cadastrar, para isso deverá clicar no botão SideBar que contém sua escolha.

2- Aqui, procure o botão atualizar / excluir na parte inferior, aqui você pode fazer essas ações.

3- Uma vez aqui, você deve selecionar um dado existente, se for a primeira vez que você entrar no SAGA, não haverá dados, então você terá que criar um.

4- Ao selecioná-lo, os campos serão preenchidos automaticamente para que você possa modificá-lo, você pode modificar todos os campos que o tipo de dado permitir.

5- Quando terminar, clique em atualizar

Etapas para excluir dados

1- Escolha a opção que deseja se cadastrar, para isso deverá clicar no botão da SideBar que contém a sua escolha.

2- Aqui, procure o botão atualizar / excluir na parte inferior, aqui você pode fazer essas ações.

3- Uma vez aqui, você deve selecionar um dado existente, se for a primeira vez que você entrar no SAGA, não haverá dados, então você terá que criar um.

4- Ao selecioná-lo, clique no botão Excluir.

5- Pedirá uma confirmação, se tiver certeza que deseja deletar clique sim.

Possíveis erros

Este tipo de erro pode ocorrer quando:

1- Você está inserindo um dado incorreto ou que o campo não suporta. Portanto, verifique o que o campo pede a você

2- Você não está conectado ao banco de dados. Verifique sua conexão, caso não entre em contato com o administrador

Quais são os benefícios de se obter um sistema como este?

  • O SAGA permite controlar de forma segura e eficiente os objetos armazenados nos seus armazéns, permitindo-lhe saber a localização, inventário e tamanho dos seus contentores
  • Proporciona um ambiente óptimo de personalização, podendo restringir as funções dos seus colaboradores, de acordo com o nível que lhes fornece
  • Você pode controlar os objetos que estão em seus containers, considerando peso, tipo, quantidade, entre outras opções, permitindo maior segurança
  • Além disso, permite que você faça um censo dos funcionários de suas instalações, a que filial está atribuída, sua posição e informações para contatá-los
  • O SAGA permite-lhe captar os contratos elaborados pelos seus colaboradores, quando foram celebrados, o cliente, o colaborador e a sucursal onde foi criado

Quais são os passos para instalar o SAGA?

  1. Prepare o seu equipamento: considere que cada um dos seus equipamentos possui os requisitos mínimos para a instalação do SAGA; Sugerimos que você use os requisitos recomendados para um melhor desempenho
  2. Obtenha o SAGA: como o SAGA é um sistema pago, você deve primeiro comprá-lo para poder instalá-lo, um arquivo executável virá aqui para a instalação do sistema
  3. Instale o sistema: após executar o arquivo o sistema estará pronto para seu primeiro uso, prossiga para a próxima etapa
  4. Primeiro uso do sistema: SAGA precisará de uma calibração para atender às suas necessidades, por isso, na primeira utilização, serão obtidos dados para uso futuro
  5. Pronto, você pode usar o SAGA com todas as suas funções!
