SAGA first use

The steps to use SAGA are very simple, which are detailed below:

1- Open the program.

2- You will be asked to start the first use, so you will have to follow the instructions that will be given to you.

3- Register your company, you can start registering containers you own.

4- Done! You can now use SAGA

Using SAGA

Steps to register

1- Choose the option in which you want to register, for this you have to click on the button of the SideBar that contains your choice.

2- Here look for the register button at the bottom.

3- Then fill in the requested information.

4- When you finish, click on save and that's it.

Steps to update a data

1- Choose the option you want to register, for this you have to click on the button of the SideBar that contains your choice.

2- Here, look for the update / delete button at the bottom, here you can do these actions.

3- Once here, you have to select an existing data, if it is the first time you enter SAGA, there will be no data, so you will have to create one.

4- When you select it, the fields will be automatically filled so that you can modify it, you can modify all the fields that the data type allows.

5- When you finish, click on update

Steps to delete a data

1- Choose the option you want to register, for this you have to click on the button of the SideBar that contains your choice.

2- Here, look for the update / delete button at the bottom, here you can do these actions.

3- Once here, you have to select an existing data, if it is the first time you enter SAGA, there will be no data, so you will have to create one.

4- When you select it, click the delete button.

5- It will ask for a confirmation, if you are sure you want to delete it, click yes.

Possible errors

This type of error can occur when:

1- You are entering an incorrect data or that the field does not support it. So check what the field asks you

2- You are not connected to the database. Check your connection, if not contact your administrator

What are the benefits of getting a system like this?

  • SAGA allows you to safely and efficiently control the objects stored in your warehouses, thus allowing you to know the location, inventory and size of your containers
  • Provides an optimal environment for personalization, being able to restrict the functions of your employees, according to the level you provide them
  • You can take control over the objects that are in your containers, considering the weight, type, quantity, among other options, allowing greater security
  • In addition, it allows you to have a census of the employees of your facilities, which branch is assigned to, their position, and information to contact them
  • SAGA allows you to capture the contracts created by your employees, when they were made, the client, the employee and the branch where it was created

What are the steps to install SAGA?

  1. Prepare your equipment: consider that each of your equipment has the minimum requirements for the installation of SAGA; We suggest you use the recommended requirements for better performance
  2. Get SAGA: since SAGA is a paid system, you must first buy it to be able to install it, an executable file will come here for the system installation
  3. Install the system: after executing the file the system will be ready for its first use, proceed to the next step
  4. First use of the system: SAGA will need a calibration to suit your needs, that is why in the first use data will be obtained for future use
  5. Ready, you can use SAGA with all its functions!
